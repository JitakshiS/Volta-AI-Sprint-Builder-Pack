Adding Features / Week 2

Adding Features

Booking / scheduling
Add a booking form with fields for name, email, phone, a date picker, and a dropdown for [service types, e.g. "yoga class, pilates, meditation"]. When submitted, show a confirmation message with the booking details.
Pricing section
Add a pricing section with 3 cards: [Basic $X/mo, Pro $X/mo, Premium $X/mo]. Each card should list 4-5 features. Add a "Most Popular" badge on the middle card.
User accounts
Add user authentication with Supabase Auth. Users should be able to sign up with email and password, log in, and see a dashboard after login. The dashboard should show [their bookings / their orders / their profile].
Database connection
Set up Supabase as the database. Create a table for [bookings / products / users] with columns for [name, email, date, status]. Connect the [booking form] so submissions save to the database.
Contact / email
The contact form doesn't send emails yet. Add EmailJS integration so form submissions go to [your@email.com]. Show a success message after sending.
Polish + Fix / Week 3

Polish + Fix

Mobile responsive
Make the entire site mobile-responsive. The navigation should collapse into a hamburger menu on small screens. All forms and cards should stack vertically on mobile.
Design consistency
Audit the entire app for design consistency. Make fonts, colors, spacing, and button styles consistent across all pages. Fix any elements that look out of place or misaligned.
Replace placeholder text
Replace all placeholder and lorem ipsum text with real copy for [your business description]. Make the headlines compelling and the descriptions clear for someone visiting for the first time.
Fix a bug
The [specific thing] is broken. When I click [button/link], [what happens] instead of [what should happen]. Fix it.
Add finishing touches
Add a favicon, update the page title to "[Your App Name]", and add meta description tags for SEO. Also add a footer with copyright, social media links, and contact info.

Final check
Do a final review of the entire app. Check for broken links, missing images, console errors, and any pages that don't look right. List everything that needs fixing and fix them.

If Things Go Wrong

Claude is confused or going in circles: Type /clear and start a fresh conversation. Paste your request again more specifically.
Claude broke something: Press Esc to stop, then type /rewind to undo the last change.
Everything is slow: Type /compact to free up memory. If still slow, do /clear and start fresh.
App won't start: Try: npm install && npm run dev
